# Joyería Chartres — Sitio Web

Sitio web estático para Joyería Chartres (joyería, artesanías y café).

## Estructura

├── index.html # Página principal
├── joyeria.html # Sección Joyería
├── artesania.html # Sección Artesanías
├── cafeteria.html # Sección Café
├── css/ # Estilos por página + base.css compartido
├── js/ # Config de Tailwind + scripts por página
└── img/ # Imágenes y video

## Tecnologías

- HTML5
- Tailwind CSS (vía CDN)
- JavaScript vanilla
- Google Fonts (Bodoni Moda, Manrope, Material Symbols)
- Sin frameworks, sin build step ni dependencias de npm
